Structure of Web Page
*********************
Structure of Web Page
*********************

Version of HTML/DOCTYPE

Root Section

		Head Section

			Metadata

		Body Section

			Main Content


DOCTYPE Representing Version of HTML.

below DOCTYPE Representing HTML 5.X

<!DOCTYPE html>


<html></html> tag, used to represent the Root Section

Root Section Divided into two sections

1) Head Section

2) Body Section

<head></head> tag, used to represent the Head section of webpage

<body></body> tag, used to represent the body section of webpage

we will define METADATA under head section

METADATA
********
	data about webpage called as METADATA

	<meta> tag, used to define the metadata


we will define main content of webpage under body section

Ex.
		images

		headings

		paragraphs

		tables

		---
		---
		---

Element
*******
	opening tag, content of webpage and closing tag called as Element

	Ex.
		<p>Hello</p>

Types of Elements

1) non container elements (empty elements)

		tags without closing tag and content of webpage called as non container elements

	Ex.
		<meta>


2) container elements
		complete element (opening tag, content of web page and closing tag) called as container element

	Ex.
		<html></html>

		<head></head>

		<title></title>

		<body></body>
